## 🔬 **生成技术细节**

### 噪音类型
1. **白噪音** - 均匀频谱（基础）
2. **粉红噪音** - 1/f 频谱（更自然，类似雨声）
3. **布朗噪音** - 1/f²频谱（更深沉，类似海浪）

### 音效生成算法

#### 雨声 (rain.wav)
```python
- 基础：粉红噪音
- 处理：低通滤波 (0.95)
- 特效：随机雨滴脉冲
```

#### 海浪 (ocean.wav)
```python
- 基础：布朗噪音
- 处理：0.1Hz 正弦波包络（模拟波浪）
- 特效：低频强调
```

#### 雷声 (thunder.wav)
```python
- 基础：静音 + 脉冲

- 特效：3-5 次随机雷声
```

#### 森林 (forest.wav)

```python
- 基础：粉红噪音
```
